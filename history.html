  • mid-1960s Navigational databases
  • IMS by IBM - had a tree-like structure within a herarchical model, used virtual records (a hack) to circumvent the tree structure.
  • 1960s Network model databases
  • 1969 CODASYL defined the Netwrok Database Language.
  • 1980s the Logical Data Model: Labeled graphs represented in graph databases.
  • 1990s Graph database improvements to support index web pages.
  • 2000s Commercial grade atomicity, consistency, isolation & durability (ACID) graph databases (Neo4j, Oracle Spatial, Graph)
  • 2010s Horizontal scaling, in-memory and columnar enhancements.
  • 2010s Multi-model databases (support relational + graph, etc)
