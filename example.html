The same query written in SQL as well as a couple of Graph DB Query Languages

In these examples we are looking of all of bobby's friends.


SQL Version 

SELECT p2.name
FROM people p1
JOIN friend ON (
    p1.id == friend.id
)
JOIN people p2 ON (
    p2.id == friend.id
)
WHERE p1.name = 'bobby'
Gremlin Version 

g.V().match(
    as("a").label().is("person"),
    as("a").out("friend_of").as("b"),
    as("b").values("name").as("c"),
    as("c").is("bobby")
).select(b")
SPARQL Version 

SELECT ?b
WHERE {
    ?a a Person .
    ?a ex:friend_of ?b .
    ?b ex:name ? c .
    FILTER(?c = "bobby")
}
